## Ferramentas de uso e estudo para criação do challenge.

    - Html, Css e javascript.
    - React.
    - Styled-Components.
    - Json Server.

## Caso queira clonar este repositório e realizar os testes, em sua máquina local



`git clone https://github.com/FabioMatech/adopetchallenge.git` 



No terminal de os seguintes comandos, para rodar os servidores :

- `npm install`

- `npm start`
- `json-server --watch db.json`
